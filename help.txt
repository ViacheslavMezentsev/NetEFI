NuGet, версия: 7.3.0.70
использование: NuGet <command> [args] [options]
Введите "NuGet help <command>" для получения справки по определенной команде.

Доступные команды:

 add                                  Добавляет данный пакет в иерархический источник. Источники HTTP не поддерживаются. Дополнительные сведения см. по ссылке: https://docs.nuget.org/consume/command
                   -line-reference#add-command.

 client-certs                         Предоставляет возможность управлять списком источников, расположенных в файлах NuGet.config.

 config                               Gets or sets NuGet config values.

 delete                               Удаляет пакет с сервера.

 eula                                 Отображение лицензионного соглашения конечного пользователя (EULA) для файла NuGet.exe

 help (?)                             Отображает общие справочные сведения и справочные сведения о других командах.

 init                                 Добавляет все пакеты из <srcPackageSourcePath> в иерархическую <destPackageSourcePath>. Веб-каналы HTTP не поддерживаются. Дополнительные сведения см. по ссылке
                   : https://docs.nuget.org/consume/command-line-reference#init-command.

 install                              Installs a package using the specified sources. If no sources are specified, all sources defined in the NuGet configuration file are used. If the configuration
                   file specifies no sources, uses the default NuGet feed.

 list                                 Отображает список пакетов из указанного источника. Если источники не указаны, используются все источники, определенные в %AppData%\NuGet\NuGet.config. Если в Nu
                   Get.config не указаны источники, используется канал NuGet по умолчанию.

 locals                               Очищает или перечисляет локальные ресурсы NuGet, такие как кэш запросов HTTP, временный кэш или глобальная папка пакетов на уровне всего компьютера.

 pack                                 Создает пакет NuGet на основе указанного файла nuspec или файла проекта.

 push                                 Отправляет пакет на сервер и публикует его.
                   NuGet's default configuration is obtained by loading %AppData%\NuGet\NuGet.config, then loading any nuget.config or .nuget\nuget.config starting from root of dr
                   ive and ending in current directory.

 restore                              Restores NuGet packages.

 search                               Выполняет поиск в заданном источнике, используя предоставленную строку запроса. Если источники не указаны, используются все источники, определенные в %AppData%\
                   NuGet\NuGet.config.

 setApiKey                            Сохраняет ключ API для заданного URL-адреса сервера. Если URL-адрес не указан, ключ API сохраняется для галереи NuGet.

 sign                                 Signs a NuGet package with the specified certificate.

 sources                              Предоставляет возможность управлять списком источников, расположенных в файлах NuGet.config.

 spec                                 Создает nuspec для нового пакета. Если эта команда запущена в той же папке, что и файл проекта (.csproj, .vbproj, .fsproj), она создаст маркированный файл nuspe
                   c.

 trusted-signers                      Предоставляет возможность управлять списком доверенных подписывающих объектов.

 update                               Обновите пакеты до последних доступных версий. Эта команда также обновляет сам NuGet.exe.

 verify                               Проверяет подписанный пакет NuGet.

Для получения дополнительной информации посетите https://docs.nuget.org/docs/reference/command-line-reference
