========================================================================
NetEFI Framework — Установка и проверка / Installation Guide
========================================================================

!! ВАЖНО / IMPORTANT !!
-----------------------
[RU] Перед распаковкой этого архива нажмите по нему ПРАВОЙ КНОПКОЙ МЫШИ,
выберите "Свойства" (Properties) и поставьте галочку "Разблокировать"
(Unblock) внизу окна. Нажмите OK и только затем распакуйте архив.
Если этого не сделать, Windows заблокирует загрузку DLL и Mathcad
не увидит функции.

[EN] Before extracting this archive, RIGHT-CLICK the .zip file, select
"Properties", and check the "Unblock" box at the bottom. Click OK,
then extract. Without this step Windows will block the DLLs and
Mathcad will not load the functions.

--- [ РУССКИЙ ] --------------------------------------------------------

УСТАНОВКА
---------

1. Скопируйте ВСЕ файлы из этого архива (кроме папки worksheets) в папку
   Custom Functions вашей установки Mathcad:

   Mathcad 15 (32-бит):
     C:\Program Files (x86)\Mathcad\Mathcad 15\userefi\

   Mathcad Prime (64-бит):
     C:\Program Files\PTC\Mathcad Prime X.X.X.X\Custom Functions\

   Для установки потребуются права администратора.

2. Перезапустите Mathcad.

3. Проверьте работу, открыв один из документов из папки worksheets,
   или набрав в рабочем листе:

     netefi("info")

   Ожидаемый результат (пример):

     "NetEFI: 32-bit, 0.5.1, 27-мар-2026 21:38:20"

МИНИМАЛЬНЫЙ НАБОР ФАЙЛОВ
-------------------------

Для работы любой пользовательской библиотеки необходимо, чтобы
в папке userefi / Custom Functions присутствовали:

   NetEFI.dll               — C++/CLI хост (обязателен)
   NetEFI.Abstractions.dll  — managed API  (обязателен)

Эти два файла уже включены в данный архив.

СОДЕРЖИМОЕ АРХИВА
-----------------

   NetEFI.dll               — C++/CLI хост
   NetEFI.Abstractions.dll  — managed API
   <sample>.dll             — пример пользовательской библиотеки
   FSharp.Core.dll          — рантайм F# (только в архиве fsharp)
   worksheets/              — документы Mathcad для проверки работы
   INSTALL.txt              — этот файл

ДИАГНОСТИКА
-----------

Лог-файл создаётся при каждом запуске Mathcad:

   %AppData%\Mathsoft\Mathcad\NetEFI.log

Если в логе появляются ошибки — убедитесь, что архив был разблокирован
ДО распаковки, и что установлен .NET Framework 4.8.

--- [ ENGLISH ] --------------------------------------------------------

INSTALLATION
------------

1. Copy ALL files from this archive (except the worksheets folder) to
   the Mathcad Custom Functions folder:

   Mathcad 15 (32-bit):
     C:\Program Files (x86)\Mathcad\Mathcad 15\userefi\

   Mathcad Prime (64-bit):
     C:\Program Files\PTC\Mathcad Prime X.X.X.X\Custom Functions\

   Administrator rights are required to copy files to Program Files.

2. Restart Mathcad.

3. Verify by opening one of the documents from the worksheets folder,
   or by typing in any worksheet:

     netefi("info")

   Expected result (example):

     "NetEFI: 32-bit, 0.5.1, 27-Mar-2026 21:38:20"

REQUIRED FILES
--------------

The following two files must be present in the userefi / Custom Functions
folder for any user library to work:

   NetEFI.dll               — C++/CLI host (required)
   NetEFI.Abstractions.dll  — managed API  (required)

Both files are already included in this archive.

ARCHIVE CONTENTS
----------------

   NetEFI.dll               — C++/CLI host
   NetEFI.Abstractions.dll  — managed API
   <sample>.dll             — sample user library
   FSharp.Core.dll          — F# runtime (fsharp archive only)
   worksheets/              — Mathcad documents to test the framework
   INSTALL.txt              — this file

DIAGNOSTICS
-----------

A log file is created on every Mathcad startup:

   %AppData%\Mathsoft\Mathcad\NetEFI.log

If errors appear — make sure you Unblocked the .zip file BEFORE
extracting it, and that .NET Framework 4.8 is installed.
